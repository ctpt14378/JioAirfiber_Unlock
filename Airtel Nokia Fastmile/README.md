# VM Setup Instructions

Follow these steps to set up and run the virtual machine:

1. Extract the provided `.7z` file in this folder.
2. Upon extraction, you will find:
   - A file named `Ubuntu_tr369_octopus.vmdk`
   - A folder named `Ubuntu_tr369_octopus`
3. Download and install **VMware** (such as VMware Workstation Player or Pro) if it is not already installed on your system.
4. Open the `Ubuntu_tr369_octopus` folder.
5. Simply double-click the `.vmx` configuration file inside the folder. This will automatically open and run the virtual machine in VMware.
